# Zulu JDK Homebrew Casks

[https://www.azul.com/downloads/zulu-community/](https://www.azul.com/downloads/zulu-community/?&os=macos&os-details=macOS&package=jdk)

## Installation:

```bash
brew tap mdogan/zulu
brew cask install <name>
```
